함수 연습

    0         1         2         
    012345678901234567890123456789
    ABCDEFGHIJKLMNOPQRSTUWXYZabcde
length : 객체(문자(열))의 길이

substring(A,B) : 문자열 발췌(A에서부터,B앞까지)

slice(A,B) : 문자열 발췌(A에서부터,B앞까지)

substr()

repalce(A,B) : 제일 앞의 A를 B로 치환

repalceAll(A,B) : A를 모두 B로 치환

toUppercase() : 대문자로 변환

toLowercase() : 대문자로 변환

concat(a,b) : a와 b문자열 결합

trim():문자열의 양쪽 공백을 무시

trimStart() : 문자열의 앞쪽 공백을 무시

trimEnd() : 문자열의 뒤쪽 공백을 무시

replaceAll(' ','') : 전체 문자열의 모든 공백을 무시

charAt() : ()위치의 한글자 추출

charCodeAt() : ()위치의 글자 아스키코드 추출

split() : 특정 문자 기준으로 잘라낸다. 이 때 잘라넨 값들은 배열변수에 기억된다.

indexOf() : 문자열 검색시에 사용 있으면 위치값, 없으면 -1 반환

lastIndexOf() : 문자열 검색시에 사용, 뒤에서 부터 찾음, 있으면 위치값, 없으면 -1 반환

search() : 문자열 검색시에 사용 있으면 위치값, 없으면 -1 반환

match() : 문자열 일치여부(있으면 문자열 그대로 반환, 없으면 공백을 반환)

include() : 문자열 포함하는지?(있으면 true, 없으면 false반환)
